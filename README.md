# Digital Pet Products Business Hub

Welcome to the central hub for **Pet Owners Dream** — your one-stop resource for digital pet products, social media content, and business information.

## 📋 What's Inside

- **Website** — Your GitHub Pages-hosted website
- **Social Media** — Content, graphics, and posting schedules for all platforms
- **Products** — Digital product catalog and descriptions
- **Marketing** — Campaigns, email templates, and promotional materials
- **Business Docs** — Plans, analytics, and financial tracking
- **Brand Guidelines** — Logo, colors, tone, and visual standards

## 🌐 Quick Links

- **Website**: [Pet Owners Dream](https://petonwersdream.com) (custom domain)
- **Social Media**: See `/social-media` directory
- **Product Catalog**: See `/docs/product-catalog.md`

## 📁 Directory Guide

```
digital-pet-products-business-hub/
├── website/              # Your GitHub Pages website
├── social-media/         # Social media content & graphics
├── docs/                 # Documentation & business info
├── marketing/            # Marketing campaigns & templates
├── business-docs/        # Financial & planning documents
└── CNAME                 # Custom domain configuration
```

## 🚀 Getting Started

1. Review the directory structure above
2. Navigate to each folder for specific content
3. Update content as needed and push changes
4. Your website updates automatically via GitHub Pages

## 📝 Notes

- Keep all sensitive information in `.gitignore`
- Update the content calendar regularly
- Use this hub to stay organized across all platforms
